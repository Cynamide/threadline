## Agent skills

### Implementation posture

This repo is agent-native. When thinking through implementation, UI, or UX, prefer agent-guided reasoning, clarification, and validation over raw user guessing wherever possible. Use direct user input only when the repo cannot be resolved confidently or a human decision is genuinely required.

Issues live in GitHub Issues for this repo. See `docs/agents/issue-tracker.md`.
