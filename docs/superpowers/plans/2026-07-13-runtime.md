# Runtime Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Ship `@threadline/runtime` as a tiny, dependency-light package that returns safe handoff wrappers with the documented development warning behavior.

**Architecture:** Keep the package to three source files: shared types, the `handoff()` implementation, and a barrel export. The public API stays object-form only, returns a callable wrapper, and never pulls in external dependencies. Development-only logging and fallback error handling live inside the wrapper so the runtime remains self-contained.

**Tech Stack:** TypeScript, Node.js 18+, pnpm workspace, Vitest or the repo's chosen test runner once the package scaffold is in place.

## Global Constraints

- Keep the package small and dependency-light
- Avoid positional arguments; the object form is the only supported shape
- Preserve a stable call shape so the AST parser can identify handoffs reliably
- `id` is required and must be a stable string literal
- `title` is required and becomes the human-readable label for the handoff
- `description` is optional but should explain the missing implementation clearly
- `fallback` is required and must be safe to run in the current UI
- async fallbacks are allowed
- warns in development mode
- stays quiet in production mode

- [ ] **Step 3: Implement the minimal runtime**

```ts
export interface HandoffOptions<T = void> {
  id: string;
  title: string;
  description?: string;
  fallback: () => T | Promise<T>;
}

export function handoff<T = void>(options: HandoffOptions<T>): () => T | Promise<T> {
  return () => options.fallback();
}
```

- [ ] **Step 2: Implement the warning branch and async-safe wrapper**

```ts
export function handoff<T = void>(options: HandoffOptions<T>): () => T | Promise<T> {
  return () => {
    if (process.env.NODE_ENV === 'development') {
      console.warn(
        `[Threadline] Handoff triggered: "${options.title}"`,
        `Description: ${options.description ?? ''}`,
        `ID: ${options.id}`,
        'This should be implemented by an engineer.'
      );
    }

    try {
      return options.fallback();
    } catch (error) {
      if (process.env.NODE_ENV === 'development') {
        console.error('[Threadline] Handoff fallback failed', error);
      }
      throw error;
    }
  };
}
```
