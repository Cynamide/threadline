# Skill Templates Context

This package defines the vocabulary for the agent-facing instruction files.

## Language

**Skill**:
A Markdown instruction file that teaches an agent how to behave in this repo.
_Avoid_: prompt pack, script, policy

**Plan**:
A sequenced breakdown of a larger UI change before any editing starts.
_Avoid_: checklist, estimate, roadmap

**Agent brief**:
The tracker-ready summary that hands deferred work to a human or downstream agent.
_Avoid_: note, comment, summary

**Boundary rule**:
A hard line around what the agent may and may not edit directly.
_Avoid_: guideline, preference, style note
